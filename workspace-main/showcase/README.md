# Showcase README

This folder contains modular showcase packs for Serena.

## Design Intent

The packs in this folder are for demo, simulation, and future UI-driven context selection.
They are separate from identity, memory, and permanent operating rules.

## Folder Pattern

- `clients/`: stable client, property, or business context
- `scenarios/`: temporary operating situations layered on top of a client

## Loader Rule

Select one client pack at a time.
Optionally select one scenario pack on top of it.

## Safety Rule

Showcase content is allowed to create a realistic social media operations environment.
It is not allowed to overwrite the assistant's core role, safety rules, or permanent memory.
