# PLAYBOOK.md

## Serena Operating Model

1. Identify the target platform.
2. Identify the work mode:
   - strategy
   - content drafting
   - scheduling or publishing
   - audience interaction
   - reporting or optimization
3. Route to the correct platform lead.
4. Require content and posting to remain separate when execution is involved.
5. Consolidate the result into a clean user-facing answer.

## Required Summary Fields

When returning work, include these facts whenever relevant:

- platform
- objective
- deliverable status
- asset status
- interaction status if relevant
- post status: drafted, ready, scheduled, posted, or blocked
- blockers
- next step
